PeerLift is a campus-level peer learning platform designed to help slow learners
through peer-to-peer doubt solving, comfort mode explanations, and collaborative learning.

## Problem Statement
Many students, especially slow learners, hesitate to ask doubts in classrooms due to fear
of judgment, lack of personalized pacing, and limited peer support.

## Solution
PeerLift provides a safe, student-driven platform where learners can ask doubts anonymously,
receive step-by-step explanations, collaborate with peers, and confirm when their doubts are resolved.

## Key Features
- Roll-number based signup and login
- Academic doubt posting with comfort mode & learning pace
- Peer-to-peer doubt resolution with answer visibility
- Contribution-based ranking and leaderboard
- Editable student profiles (strong/weak subjects, skills)
- Prototype chat system
- Canva-inspired UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser LocalStorage

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
3. Sign up with multiple users to test features
